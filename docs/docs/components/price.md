Renders the price if the ad is from a paid service/app.

```jsx
import { PriceView } from "react-native-admob-native-ads";

<PriceView
  style={{
    fontWeight: "bold",
    fontSize: 10,
  }}
/>;
```

## props

Inherits all the props from Text Component.
